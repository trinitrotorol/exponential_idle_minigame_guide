Exponential Idle ミニゲーム盤面ソルバー

現在の盤面を入力して、次にどのマスを操作するかを確認できます。

矢印パズル

入力では、各マスをゲーム内の現在向きに合わせます。解答後は、ゲーム内で押すマスを順番に確認します。

使う前のおすすめ設定
  • Visual Scheme: Greyscale + Numbers
  • Animation: 無効
  • 盤面入力中は、各マスを現在の向きに合わせるだけです
  • ゲーム内のように周囲のマスは回転しません

入力中のタップは入力だけを変更します。解答ステップの強調マスはゲーム内で押すマスです。

解法

15パズル

数字を空白へ移動しながら順番通りに並べます。入力では0または空欄を空白として扱います。

使う前のおすすめ設定
  • Visual Scheme: Fringe
  • Hover/Slide Control: 有効
  • Animation: 無効
  • 空白はこのサイトでは空欄として表示します

Hardは5x5です。最適解ではなく、層を固定して解く実用手順を出します。

解法

トーラスパズル

行または列を巡回シフトして、左上から昇順に並べます。操作表示の行・列番号は1始まりです。

使う前のおすすめ設定
  • Visual Scheme: Insertion
  • Hover/Slide Control: 有効
  • Animation: 無効
  • 行・列番号はこのサイトでは1始まりで表示します

Easyは3x3、Mediumは5x5、Hardは6x6です。
